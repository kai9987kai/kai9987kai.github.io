# No Instructions For Hallucinogens

An educational neuroscience writing project about how hallucinogens/psychedelics affect the brain across scales:

- receptor biophysics
- cortical/thalamic circuitry
- large-scale network dynamics
- plasticity and therapeutic windows

This project is intentionally non-instructional. It does not include dosing, sourcing, or "how to use" guidance.

## Project Goal

Turn the existing discussion into a rigorous, readable long-form piece (or series) that connects:

1. Molecular pharmacology
2. Electrophysiology and brain networks
3. Behavior and subjective experience
4. Therapeutic plasticity and open research questions

## Current Files

- `drafts/extended_frontiers.md` - expanded continuation of the ideas you pasted
- `drafts/ch01_from_receptor_to_perception.md` - first chapter draft (receptor -> cortex -> early subjective changes)
- `drafts/ch02_how_the_thalamus_loses_the_room.md` - second chapter draft (thalamic gating / CSTC framing)
- `research/claim_checklist.md` - statements to verify/cite before publishing
- `research/source_map.md` - primary-source map for major claims and corrections
- `project-plan.md` - execution plan for building this into a finished project
- `mkdocs.yml` + `docs/` - docs/book-style publication scaffold (Markdown site)

## Editorial Guardrails

- No drug-use instructions
- Distinguish evidence from hypothesis
- Flag contested claims clearly
- Prefer primary literature for factual claims

## Quick Start (Docs Scaffold)

If you want to preview this as a docs/book-style site:

1. Install `mkdocs` (and optionally a theme)
2. Run `npm run docs:dev` (or `npm run docs:sync` then `mkdocs serve`)
3. Open the local URL shown in the terminal

The `docs/chapters/` copies are publishing-oriented snapshots. The current writing source of truth is still `drafts/`.
