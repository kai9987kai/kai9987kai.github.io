# Overview

## Core Thesis

Hallucinogens reveal that consciousness depends on controlled constraint.

When receptor-level perturbations loosen those constraints, the brain can become temporarily:

- more cross-connected
- less predictively stable
- more sensitive to both signal and noise
- more updateable

The same shift can produce hallucination, insight, fear, or therapeutic change depending on context and what is learned while the system is in that state.

## Scope (Current)

Primary focus:

- classical serotonergic psychedelics (`LSD`, `psilocybin/psilocin`, `DMT`)

Secondary/comparative references (handled cautiously):

- `MDMA`
- `ketamine`
- `ibogaine`

## Evidence Standard

- Prefer primary literature for factual claims
- Use reviews for framing and terminology
- Label computational frameworks (for example `REBUS`) as frameworks, not settled proof
- Avoid collapsing preclinical plasticity results into direct human clinical claims
