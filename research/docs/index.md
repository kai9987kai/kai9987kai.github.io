# No Instructions For Hallucinogens

Educational neuroscience project exploring psychedelics across scales:

- receptor pharmacology
- cortical/thalamic circuitry
- oscillations and network dynamics
- plasticity and therapeutic windows

This project is non-instructional. It does not include dosing, sourcing, or use guidance.
