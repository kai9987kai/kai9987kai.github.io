AR Fire Lab V3

This version keeps Hiro marker AR as the stable default and adds a WebXR-ready design path for markerless placement later.

  • Tracking: smoothed marker pose, lock/loss telemetry, stability score, slow movement guidance.
  • Simulation: adaptive particle budget, wind, turbulence, sparks, smoke/ember/neon modes, thermal pulse lighting.
  • Tools: local presets, debug JSON export, snapshot attempt, scan marker shortcut, mobile-friendly controls.
  • Best use: open over HTTPS or localhost, allow camera, use a flat printed Hiro marker in good light.

Console API: ARFireLabV3.burst(), ARFireLabV3.setMode('plasma'), ARFireLabV3.exportDebug()
