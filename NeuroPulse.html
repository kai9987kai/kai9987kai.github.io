Invention Summary what’s “ground-breaking” here

This file isn’t just a neuron simulator. It’s a neuroengineering workbench:
  • CableLab: a spatially distributed HH axon (N compartments) with adjustable axial coupling (gAx).
  • Kymograph Propagation View: real-time space-time voltage map (traveling wave visualization).
  • NeuroProtocol DSL: script experimental stimulation sequences (dc, pulse trains, ramps, chirps, noise) with spatial targeting.
  • CSV Export: record + export datasets for analysis / coursework / ML pipelines.
